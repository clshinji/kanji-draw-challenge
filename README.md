# かんじれんしゅう

小学2年生向けの漢字書き取り練習アプリです。マイクラ風のドット絵デザインで、楽しみながら書き順を覚えられます。

## アプリにアクセス

iPadのカメラでQRコードを読み取ってください。

## 主な機能

- **書き順れんしゅう** — 一画ずつガイドが表示され、正しい書き順で練習できます
- **レベルアップ** — 練習するとけいけんちが貯まり、レベルが上がります
- **バッジコレクション** — 漢字をマスターするとバッジがもらえます
- **おにわ育成** — 練習を続けるとおにわの植物が育っていきます

## 対象デバイス

iPad + Apple Pencil での利用を想定しています。

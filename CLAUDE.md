# 小学生向けの漢字練習アプリ

- 子どもが楽しんで漢字の練習ができるアプリを作りたい
  - 今は小学２年生のを作っているが、将来的には1〜6年生までの漢字を網羅したい
  - 漢字の書き順が苦手なので、ストレス無く書き順を学べるようにしたい
  - 継続して学べることが必要なので、挫けないように、楽しく遊びながら学習を進めて欲しい
- 子どもが利用するデバイスは **iPad** です。
- **AIが作ったような、バイブコーディングでよく見るデザイン** は **ゼッタイにNG** です！！！！
- **開発で得られた経験はドキュメント化** して、チームの資産として蓄積します！
